The menu


The menu is a left-side panel that can be opened with a hidden button.
That hidden button can be reached by clicking the top left side of the window screen.

If you didnt understand that, it's the top left part of this white-background documentation page.

If you're still dumb and don't know where it is, just tap the Z key on your keyboard to open up instead.

But... what is this? What does everything mean?

Volume means the required volume for your character to start using the talking animation. Below, you have your current microphone volume saying "right now:" and a number.

Try speaking into your microphone. It gets higher, right? Talk for a few seconds and calculate by mind what you think the value should be. (by the way, the default is 11 because it works fine with that on mine)

Okay okay okay, but now, what about the URLs? What are they?

The first URL is the FBX file of your character with an idle animation, without talking.

The second URL is the FBX file of your character with a talking animation.

(in conclusion, you can read the Character section for more info)

Using a character



Now, what are the so called "Other" settings? 0F0? BG Image?
The first textbox (defaulted to 0F0) is the hex code (#RGB or #RRGGBB) of the background color that david3VT will use. Think of it as a chroma key color also known as a greenscreen.

The second textbox (BG Image URL) is where you put the URL of a background image if you want to directly replace like chroma key does without any imperfections. If you want a 3D model you should just be able to add it to your character separate from the animations making it look like maybe you are in a house.

If you simply just want something like a regular VTuber to put it over a screen capture in OBS, change the first textbox's hex color to the chroma key color you wish and filter it on OBS (for transparency).

The plain shading checkbox marks if you want flat shading or not (see Lighting section).

The textbox next to it is the color you want the "scene" light to be in something like 0xRRGGBB hexadecimal.

Finally, use "Close" to close the menu and/or "Apply" to apply the parameters.
